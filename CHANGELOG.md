## [0.1.0] - 2026-06-13

### Added

- Open, display, and save `.xlsx` workbooks with sheet, cell, style, formula,
  and layout data.
- Spreadsheet workspace with top chrome, formula bar, grid canvas, side rails,
  bottom sheet tabs, file drag-and-drop, and File menu actions.
- Interactive grid navigation, selection, inline editing, formula-bar editing,
  merged cells, hidden rows/columns, and styled cell rendering.
- Column and row resizing with drag handles, double-click auto-fit, and workbook
  save support.
- Workbook state management for active sheet, selected range, scroll position,
  zoom, dirty tracking, and unsaved-change prompts.
- Appearance controls for system, light, and dark modes, including persisted
  preferences and dark-theme cell text contrast adjustment.
- Cross-platform close handling and dirty-prompt normalization for Linux and
  Windows.
